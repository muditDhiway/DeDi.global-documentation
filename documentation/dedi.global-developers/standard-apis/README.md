# Standard APIs

The DeDi Protocol defines a minimal set of standard APIs that make universal discoverability possible. Any system implementing this protocol must expose these APIs to be considered DeDi-compliant.

Currently, three API types are defined:

#### 1. Lookup API

#### 2. Query API

#### 3. Version API

Each serves a distinct role in navigating and retrieving hierarchical data.

#### Data Population

While the DeDi Protocol standardizes data discoverability, it does not dictate data population mechanisms.
